# support-data-presentations

This repository contains the support data used in the oral presentations I gave at conferences and workshops.

## BEA2025

Oral talk presented at the [20th Workshop on Innovative Use of NLP for Building Educational Applications](https://sig-edu.org/bea/2025), entitled "You Shall Know a Word’s Difficulty by the Family It Keeps: Word Family Features in Personalised Word Difficulty Classifiers for L2 Spanish" (see also [corresponding paper](https://aclanthology.org/2025.bea-1.24/)).

## EUROCALL2023

## LATILL2025
